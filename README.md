# OC Fate Generator

An interactive generator for building male OC setups and fate arcs through a compositional axis system.

## Overview

This project lets you combine world, motive, power, relationship, and aesthetic axes to generate character setups and narrative skeletons with stronger internal logic than flat trope lists.

Rather than choosing from generic labels alone, the generator works through interacting dimensions such as:

- world constraints
- embodiment
- power type
- role stance
- motive
- choice under pressure
- expression style
- empathy mode
- sanity / stability
- power dynamic
- gaze / relational view
- love authenticity
- weakness trigger
- time pressure
- authority scope
- cost
- ending form
- aesthetic palette

## Live Demo

**http://www.lnln.fun/oc-gen/**

Open the page, select at least three axes, optionally add extra preferences, and generate.

## How It Works

### Axis System

The generator uses 18 compositional axes. Each axis represents a dimension of character or narrative design:

| Code | Axis | Description |
|------|------|-------------|
| **W** | World | Setting constraints, genre rules, metaphysical limits |
| **B** | Body | Embodiment, physical form, species, modifications |
| **P** | Power | Ability type, source, scope, limitations |
| **R** | Role | Social position, narrative function, stance |
| **M** | Motive | Core drive, what they want and why |
| **C** | Choice | Decision under pressure, moral inflection points |
| **E** | Expression | Communication style, emotional vocabulary |
| **J** | Judgment | Empathy mode, how they read others |
| **S** | Sanity | Mental stability, reliability of perception |
| **D** | Dynamic | Power structure in relationships |
| **V** | View | Gaze / relational perspective |
| **L** | Love | Authenticity of affection, attachment style |
| **A** | Achilles | Weakness trigger, vulnerability |
| **T** | Time | Temporal pressure, urgency, lifespan constraints |
| **G** | God-mode | Authority scope, reality-editing power |
| **X** | eXchange | Cost, sacrifice, transactional logic |
| **F** | Finale | Ending form, resolution type |
| **Palette** | Aesthetic | Visual / tonal style, genre flavor |

### Generation Modes

The output structure changes based on which axes you select:

- **Opening Mode**: When F / X / T / G are **not** selected → generates a character setup plus opening scene
- **Timeline Mode**: When any of F / X / T / G **are** selected → generates a broader fate skeleton with arc progression

# OC 命运生成器

一个通过多轴系统生成男性 OC 设定与命运骨架的交互式工具。

## 项目概述

这个项目允许你通过世界、动机、力量、关系与美学等多个维度的组合，生成内部逻辑更完整的角色设定与命运骨架，而不是停留在平面的 trope 标签堆砌。

它不是单纯从几个关键词里随机抽卡，而是基于一组可组合、可联动的轴来生成角色：

- 世界约束
- 躯体形态
- 力量类型
- 立场关系
- 动机支柱
- 压力下的抉择
- 表达方式
- 共情模式
- 心智稳定性
- 权力结构
- 凝视方式 / 关系视角
- 爱的真伪
- 软肋触发点
- 时间压力
- 权限范围
- 代价
- 终局形态
- 美学调色板

## 在线试玩

**http://www.lnln.fun/oc-gen/**

打开页面后，至少选择 3 个轴，可选填写补充偏好，然后直接生成。

## 使用方式

### 轴系统

生成器使用 18 个组合轴。每个轴代表角色或叙事设计的一个维度：

| 代码 | 轴 | 说明 |
|------|-----|------|
| **W** | World（世界） | 世界观约束、类型规则、形而上限制 |
| **B** | Body（躯壳） | 躯体形态、物种、改造 |
| **P** | Power（力量） | 能力类型、来源、范围、限制 |
| **R** | Role（立场） | 社会位置、叙事功能、角色 stance |
| **M** | Motive（动机） | 核心驱动力，想要什么以及为什么 |
| **C** | Choice（抉择） | 压力下的决策、道德转折点 |
| **E** | Expression（表达） | 沟通风格、情感词汇 |
| **J** | Judgment（共情） | 共情模式、如何解读他人 |
| **S** | Sanity（心智） | 精神稳定性、感知的可靠性 |
| **D** | Dynamic（权力） | 关系中的权力结构 |
| **V** | View（凝视） | 凝视方式 / 关系视角 |
| **L** | Love（真伪） | 爱的真伪、依恋模式 |
| **A** | Achilles（软肋） | 软肋触发点、脆弱性 |
| **T** | Time（时间） | 时间压力、紧迫性、寿命限制 |
| **G** | God-mode（神权） | 权限范围、现实编辑能力 |
| **X** | eXchange（代价） | 代价、牺牲、交易逻辑 |
| **F** | Finale（终局） | 终局形态、结局类型 |
| **Palette** | 调色板（美学） | 视觉 / 音调风格、类型风味 |

### 生成模式

输出结构根据所选轴而变化：

- **开场模式**：未选择 F / X / T / G 时触发 → 生成角色设定与开场场景
- **时间线模式**：选择了 F / X / T / G 任一时触发 → 生成更完整的命运骨架与弧线进展
