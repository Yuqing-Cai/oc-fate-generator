# 轴设计指南

OC Fate Generator 的轴系统设计文档，包含轴定义、联动规则、以及生成建议。

---

## 轴系统总览

生成器使用 17 个组合轴 + 1 个调色板轴。每个轴代表角色或叙事设计的一个维度：

| 代码 | 轴名 | 说明 | 选项数 |
|------|------|------|--------|
| **W** | World（世界） | 世界观约束、外部阻力 | 6 |
| **B** | Body（躯壳） | 躯体形态、物种 | 3 |
| **P** | Power（力量） | 力量量级，从凡人到造物主 | 3 |
| **R** | Role（立场） | 与秩序的关系 | 3 |
| **M** | Motive（动机） | 核心驱动力 | 4 |
| **C** | Choice（抉择） | 压力下的决策模式 | 3 |
| **E** | Expression（表达） | 沟通风格、恋爱手感 | 5 |
| **J** | Judgment（共情） | 共情模式 | 3 |
| **S** | Sanity（心智） | 精神稳定性 | 3 |
| **D** | Dynamic（权力） | 关系中的权力结构 | 3 |
| **L** | Love（关系认知） | 他眼里你是什么 + 爱的真伪，以进化弧线呈现 | 5 |
| **A** | Achilles（软肋） | 致命弱点 | 3 |
| **H** | Heroine（她的位置） | 女主的能动性与角色定位 | 4 |
| **T** | Time（时间） | 时间压力 | 4 |
| **X** | eXchange（代价） | 牺牲与代价 | 3 |
| **F** | Finale（终局） | 关系最终形态 | 4 |
| **Palette** | 调色板 | 文本质感 | 8 |

### v2 变更说明

- **合并 V（凝视）+ L（真伪）→ L（关系认知）**：原 V 轴和 L 轴高度重叠（V2"你是药" ≈ L2"爱你的功能"），合并后以"起点→终点"弧线呈现，消除矛盾选择
- **合并 P（力量）+ G（神权）→ P（力量）**：原 P 管类型、G 管范围，属于同一维度的不同切面，合并为统一的"力量量级"
- **新增 H（她的位置）**：原系统完全围绕男主，女主被默认为被动接收者。H 轴定义女主能动性
- **精简 Palette**：合并相似选项（清冷留白+冷感疏离→冷调克制，浓艳炽烈+华丽危险→炽艳锋芒），从 10 项降至 8 项
- **补全联动规则**：新增 B→T、S→E/J、R→D、H→D/L、P→X 等联动

## 轴详细定义

### W - World（世界）

**说明：** 故事发生在什么样的世界里，外部阻力是什么。

| 选项 | 名称 | 描述 |
|------|------|------|
| W1 | 铁律之笼 | 适用于朝堂/军队/财阀/严格制度。核心矛盾：个人意志 vs 集体规训。 |
| W2 | 废墟之野 | 适用于末日/战乱/废土。核心矛盾：生存 vs 情感。 |
| W3 | 虚无之海 | 物质富足但精神空洞。核心矛盾：活着的意义是什么。 |
| W4 | 暗面之城 | 表面正常，底下有东西（都市奇幻/怪谈/双面生活）。 |
| W5 | 未知之境 | 星际/异世界/探索。核心矛盾：共同面对无法预测的未来。 |
| W6 | 修罗之场 | 选秀/科举/商战/零和博弈。核心矛盾：竞争下的感情。 |

### P - Power（力量）

**说明：** 他的力量量级。同时决定气场、能力范围、和代价的重量。

| 选项 | 名称 | 描述 |
|------|------|------|
| P1 | 凡人之力 | 在世界规则框架内运作。能救局部，不能改写命运。 |
| P2 | 规则之上 | 能修改法律/操控市场/影响局部因果。强大但有连锁反应。 |
| P3 | 造物之权 | 接近神的能力，可改世界底层法则。全能意味着全责。 |

### L - Love（关系认知）

**说明：** 他眼里你是什么 + 他爱的到底是不是你。以"起点→终点"弧线呈现。

| 选项 | 名称 | 弧线 |
|------|------|------|
| L1 | 猎物→真心 | 从不把你当人 → 转折点：开始当人看 → 爱上真实的你 |
| L2 | 药物→独立 | 依赖你（止痛药/锚点）→ 学会独立 → 自由前提下选择留下 |
| L3 | 影子→放手 | 爱脑补的你/逝去之人影子 → 你展现真实一面 → 杀死心里的"完美你" |
| L4 | 劫数→和解 | 恨自己因你软弱 → 内耗 → 接受"爱让人脆弱≠变弱" |
| L5 | 始终如一 | 从始至终爱真实的你，无进化弧线，考验外部张力 |

### H - Heroine（她的位置）

**说明：** 女主的能动性和角色定位。直接影响 D 轴（权力）和 L 轴（进化速度）。

| 选项 | 名称 | 描述 |
|------|------|------|
| H1 | 独立对手 | 有自己的目标和力量，势均力敌，谁先动心谁先输 |
| H2 | 共生搭档 | 互补，各有所长，"一起比分开强" |
| H3 | 暗处破局 | 看似弱势但掌握核心矛盾的关键 |
| H4 | 柔软之刃 | 不强大，但存在本身改变了他 |

### D - Dynamic（权力）

**说明：** 关系中的权力结构。几乎不可能始终不变。

| 选项 | 名称 | 描述 |
|------|------|------|
| D1 | 他在上位 | 师尊/帝王/造物主。看点是他低头的时候。 |
| D2 | 他在下位 | 侍卫/奴隶/保镖。"以下犯上"的张力。 |
| D3 | 势均力敌 | 宿敌/同僚。赢家是先动心的那个。 |

## 轴联动规则

### 核心联动

| 联动 | 规则 |
|------|------|
| W → C | 世界越残酷，抉择越痛 |
| M → C | 动机驱动抉择，除非被爱动摇 |
| E + J | 沟通模式：E 是输出，J 是输入，错配产生误会 |
| T + F | 时间压力如何导向终局 |
| X ↔ M | 代价必须与动机呼应 |

### v2 新增联动

| 联动 | 规则 |
|------|------|
| B → T | 凡人身体 + 非人对象 = 天然寿命差 |
| S → E, J | 崩坏的心智无法维持精密表达或可靠共情 |
| R → D | 秩序守卫者倾向上位，被抛弃者倾向下位 |
| P → X | 力量越接近造物之权，代价越可能是存在级的 |
| H → D | 她的能动性重新定义权力格局 |
| H → L | 她的能动性影响关系认知的进化速度 |
| L + A | 他的弱点是否在你手里，取决于他怎么看你 |

## 生成模式

### 开场模式

当未选择 F / X / T 时触发。

**输出：** 角色设定 + 开场场景

**适合：** 细腻的角色塑造，建立第一印象

### 时间线模式

当选择 F / X / T 任一轴时触发。

**输出：** 完整命运骨架 + 弧线进展

**适合：** 长篇规划，命运走向预览

## 最佳实践

### 推荐轴组合（低风险）

| 组合 | 特点 |
|------|------|
| L5 + D3 + H1 | 健康平等关系，双方都有能动性 |
| M3 + S1 + J2 | 非创伤动机驱动 |
| B1 + E3 + H2 | 日常叙事，共生搭档 |

### 谨慎使用的组合

| 组合 | 风险 | 缓解建议 |
|------|------|---------|
| L2 + D1 + H4 | 功能化依赖 + 上位者 + 被动女主 | 确保 L2 的弧线完整推进到"独立"终点 |
| L3 + A2 | 影子文学 + 系于一人 | 确保 L3 弧线推进到"放手"，承认她是独立个体 |
| S3 + E2 | 崩坏心智 + 风流撩拨 | 不合理，建议改为 E1 或 E3 |
| P3 + X1 | 造物之权 + 降格 | 合理但需写清降格的具体代价和不可逆性 |
